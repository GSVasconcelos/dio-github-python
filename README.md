# Desafio GitHub com Python da DIO
#### Desafio da DIO do curso de GitHub Certification.

Este repositório foi criado para fins de conclusão duma unidade do curso de gitHub certification da DIO, cujo desafio é solucionar problemas usando python e subi-los remotamente.

## Arquivos python criados e suas funções:
- `concat_strings.p`: concatenar duas strings inseridas pelo usuário;
- `rep_strings.py`: repetir uma string que o usuário inseriu pela quantidades de vezes que ele desejar;
- `operacoes.py`: o usuário digita dois números, depois seleciona uma operação matemática e o programa efetua essa operação com os números inseridos;
- `par_impar.py`: o programa verifica se o número que o usuário digitar é par ou ímpar;
- `media.py`: o programa efetua uma média aritimética com dois números que o usuário digitar;
- `polindromo.py`: o programa verifica se a palavra que o usuário digitar é um polindromo.
